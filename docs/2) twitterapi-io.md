# twitterapi.io

[twitterapi.io](https://twitterapi.io/) is used to query X/Twitter posts from various X accounts. This service is minimal in cost. Once an account is created an `API_KEY` to use their APIs is also created.

## Setup

- Add the `twitter_api_io_key` from [twitterapi.io](https://twitterapi.io/) to the `config.json` file for the `social-media-daily` repo.

## Payments

Make a single payment for API use. The $20 USD amount should last more than a year. Payment with USDC works but be patient as it takes time to process.
